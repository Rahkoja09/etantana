# e_tantana

Une application mobile intuitive conçue pour simplifier la gestion des ventes en ligne et la logistique.

### Objectif du MVP
Tester et valider l'intégration d'une **mini-imprimante thermique externe** pour automatiser la génération de :
* **Factures clients** (détaillées et professionnelles).
* **Étiquettes de colis** (pour faciliter la livraison).

>  **Version actuelle :** Android Only

## Fonctionnalités clés
* **Gestion des commandes :** Suivi en temps réel des ventes.
* **Impression Thermique :** Connexion Bluetooth avec support des rouleaux 80mm.
* **Export PNG :** Impression rapide des factures via IPrint.
